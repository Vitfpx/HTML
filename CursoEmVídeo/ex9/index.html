Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>

Nesta frase tenho um termo em destaque usando a tag <strong>(semântica, o Strong passa uma impressão de maior urgência naquilo que ele destaca)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em>(semântica)

Texto marcado

Podermos criar também um texto marcado usando a tag <mark>

A cor do mark não é global, ou seja, aqui continuaria padrão

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste trecho usando <big> e <small>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Ele é feito utilizando a tag <del>

Texto insetido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Faça isso com a tag <ins>

Existem também o sublinhado com a tag <u> (Não semântico)

Texto sobrescrito

Para inserir coisas do tipo x20+3. Isso se faz com a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O. Consiga esse feito com a tag <sub>
